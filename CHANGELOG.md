## [0.2.9] - 2024-11-15

### Added
- OpenTelemetry-native observability with comprehensive tracing
- Three integration patterns: wrapper functions, @observe decorator, and native SDK
- Intelligent routing across 250+ LLM providers
- Semantic caching with vector similarity matching
- Cost optimization (30-50% reduction in LLM costs)
- Real-time analytics and evaluation framework
- Support for Python 3.9, 3.10, 3.11, and 3.12
- Comprehensive testing with pytest, integration tests, and security scans
- Manual test scripts for interactive testing

### Changed
- Migrated to setuptools build system with pyproject.toml
- Enhanced configuration management with Pydantic models
- Improved authentication with project-scoped API keys

### Fixed
- Various bug fixes and stability improvements
